===============================================================================
        LongVA 完整评测 - 快速开始指南
===============================================================================

📋 评测配置
-----------
• 模型数量: 3 (LongVA-Temporal-v1, LongVA-Temporal-v2, LongVA-7B)
• 数据集数量: 10 个主流benchmarks
• 评判方式: OpenAI API
• 断点续传: ✓ 支持（自动跳过已完成的评测）

===============================================================================
🚀 第一步：配置 OpenAI API Key
===============================================================================

编辑配置文件：
    cd VLMEvalKit
    nano .env

修改这一行（替换为你的实际API key）：
    OPENAI_API_KEY=sk-your-actual-openai-key-here

===============================================================================
🎯 第二步：运行完整评测
===============================================================================

一键启动（推荐）：
    cd VLMEvalKit
    bash ../eval_all_models.sh

后台运行（推荐用于长时间评测）：
    cd VLMEvalKit
    nohup bash ../eval_all_models.sh > eval.log 2>&1 &

查看后台运行进度：
    tail -f eval.log

===============================================================================
⏱️  预计时间
===============================================================================

• 单个模型单个数据集: 10-60分钟
• 完整评测（3模型 × 10数据集）: 6-12小时

===============================================================================
🔄 断点续传
===============================================================================

如果评测中断，直接重新运行即可自动继续：
    cd VLMEvalKit
    bash ../eval_all_models.sh

VLMEvalKit 会自动跳过已完成的评测！

===============================================================================
📊 查看结果
===============================================================================

结果保存位置：
    VLMEvalKit/longva_results_YYYYMMDD_HHMMSS/

快速查看所有结果：
    cd VLMEvalKit
    find longva_results_* -name "*.csv" | sort

查看日志：
    cat eval_progress_*.log

===============================================================================
🔧 自定义评测（可选）
===============================================================================

评测单个模型：
    cd VLMEvalKit
    python run.py --data MMBench_DEV_EN --model LongVA-Temporal-v1 --verbose

评测特定数据集：
    cd VLMEvalKit
    python run.py \
        --data MMBench_DEV_EN SEEDBench_IMG MME \
        --model LongVA-Temporal-v1 LongVA-Temporal-v2 LongVA-7B \
        --verbose
